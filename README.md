# Respiratory-Ailment-Classification-AI
A machine learning model trained on .wav files of COPD, Healthy, URTI, Bronchiectasis, Pneumonia, Bronchiolitis, Asthma, and LRTI to propose probabilities of diagnoses. 

Utilizes the Respiratory Sound Database from kaggle.com: https://www.kaggle.com/datasets/vbookshelf/respiratory-sound-database?resource=download
