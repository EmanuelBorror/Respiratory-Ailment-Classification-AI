# Respiratory-Ailment-Classification-AI
A machine learning model trained on .wav files of COPD, Healthy, URTI, Bronchiectasis, Pneumonia, Bronchiolitis, Asthma, and LRTI breathing to propose probable diagnoses.  

See the dataset_download file to download the dataset for training with this script. 

Utilizes the Respiratory Sound Database from kaggle.com: https://www.kaggle.com/datasets/vbookshelf/respiratory-sound-database?resource=download
